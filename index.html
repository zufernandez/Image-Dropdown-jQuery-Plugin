Image Dropdown Plugin

Image Dropdown Plugin is a simple jQuery plugin to help you show images on a select instead of using old dropdown text lists.

How to use

  1. Add a reference to jQuery
  2. Add a reference to imagedropdown1.1.js and imagedropdown.css
  3. Write a 'normal select'. Select element MUST HAVE a name. If you have more than one select on your page make sure they have different name. Image to show will be the text on option. 
    <select name="eg1">
<option value ="111">images/1.jpg</option>
<option value ="222">images/2.jpg</option>
<option value ="333">images/3.jpg</option>
</select>
  4. Call the plugin 
    $('.myselect').imagedropdown({  
	width :60,		//default: 100
	listheight: 180 //default: 250
});
  5. Read the selected value the usual way (read the value on the select element)

Example

I'm making the select visible so you can see how the value is changed when an image is clicked.

If an image has a selected value, this will be reflected on the image dropdown list.I'm selecting image 3.
